# üß± EASY FUEL ‚Äî USER DATA STRUCTURE EXPLANATION

Each person or company that signs up will fall under one of these roles:
**Customer, Driver, Supplier, or Admin.**
All roles share some basic information, and then each role adds its own details.

## üßç‚Äç‚ôÇÔ∏è SHARED PROFILE FIELDS (everyone)

These are stored for every account, regardless of role.

| Field                                       | Description                                                                                   |
| ------------------------------------------- | --------------------------------------------------------------------------------------------- |
| **user_id**                                 | Unique ID automatically assigned to every user.                                               |
| **role**                                    | The type of account (Customer, Driver, Supplier, Admin).                                      |
| **first_name / last_name / full_name**      | Basic user name information.                                                                  |
| **email**                                   | Used for login and communication.                                                             |
| **phone_country_code / phone_number**       | Phone number for verification or contact.                                                     |
| **profile_photo_url**                       | Picture of the user or company logo.                                                          |
| **approval_status**                         | Shows if the account is *Pending*, *Approved*, *Rejected*, *Needs more info*, or *Suspended*. |
| **approval_reason**                         | Explains why an account was approved or rejected.                                             |
| **approved_by / approved_at**               | Records which admin approved the user and when.                                               |
| **is_active**                               | Marks whether the user can currently access the platform.                                     |
| **notes**                                   | Free text field for admin notes or comments.                                                  |
| **address**                                 | Stores address information (street, city, province, postal code, country).                    |
| **created_at / updated_at / last_login_at** | Tracks when the profile was created, last updated, and last logged in.                        |
